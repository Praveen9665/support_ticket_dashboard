# Support Ticket Dashboard

A premium, interactive dashboard built with React and Tailwind CSS.

##  Key Features
- **Real-time Updates**: Automatically refreshes ticket data.
- **Advanced Filtering**: Search by ID, customer, or subject.
- **Optimistic UI**: Instant status updates with background synchronization.
- **Responsive Design**: Clean and professional look on any screen.

##  Getting Started

1. **Environment Config**:
   Create a `.env` file with:
   ```env
   VITE_API_BASE_URL=http://localhost:5000/api
   ```

2. **Install & Run**:
   ```bash
   npm install
   npm run dev
   ```

##  Authentication
The system uses a demo login flow:
- **Email**: Any valid email format.
- **Password**: Minimum 6 characters.

##  Tech Stack
- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **HTTP Client**: Axios
